Project: MyStudent (working title) — Student Schedule Management Web App
Concept A web app that ingests all classes, homework, readings, and assignments and generates a day-by-day study schedule. Not just due dates — it looks ahead across the next week and allocates time intelligently (e.g. "spend 1 hour on this coding assignment today because it's due in 5 days").
Tech Stack
TypeScript + React (frontend/UI)
localStorage or Supabase (data persistence)
Start scheduling logic prototyped in Python first, then port to TypeScript
Developer Background
3 years coding experience
Proficient in Python, C, C++, Java
Some JavaScript experience
Taking a 4-credit TypeScript course in the fall — this project is a good head start
Building a web app rather than iOS to reduce complexity and leverage existing skills
Build Strategy
Start with a Python script to nail the scheduling logic (input: classes, due dates, estimated hours → output: daily plan)
Once logic is solid, rebuild in TypeScript/React as a full web app
iOS can come later if desired
